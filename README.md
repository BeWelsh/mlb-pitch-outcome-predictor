# mlb-pitch-outcome-predictor

To run locally:
1. Download the 2019_pitches.csv from https://www.kaggle.com/datasets/pschale/mlb-pitch-data-20152018?resource=download&select=2019_pitches.csv
2. Download the 2019_atbats.csv fromhttps://www.kaggle.com/datasets/pschale/mlb-pitch-data-20152018?resource=download&select=2019_atbats.csv
3. Move the csv files into the root of this repository
4. Pat yourself on the back
